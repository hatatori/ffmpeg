Set volume video

ffmpeg -i input.mp4 -af volume=1.5 output.wav
ffmpeg -i input.mp4 -af volume=1.5 output.wav
ffmpeg -i input.mp4 -af volume=0.5 output.wav

Set volume audio

ffmpeg -i input.mp3 -va volume=1.5 output.mp3
ffmpeg -i input.wav -va volume=1.5 output.wav
ffmpeg -i input.wav -va volume=0.5 output.wav

Remove volume

ffmpeg -i input.mp4 -an output.mp4

Merge audio+video

Convert to aac

ffmpeg -i input.mp4 audio.aac

Merge audio + video

ffmpeg -i input.mp4 -i audio.aac output.mp4
